## 🏛️ Executive Summary

OmniMind is a state-of-the-art, **Expertise-Aware Orchestration Engine** designed to solve the fragmentation of modern AI. Instead of interacting with isolated models, OmniMind provides a singular, high-performance interface that dynamically routes, synthesizes, and grounds intelligence across multiple specialist pools.

Built with the **Liquid Glass Design System**, OmniMind delivers a premium, research-first experience that prioritizes visual clarity, reasoning transparency, and autonomous discovery.

---

## 🧠 The Brain Router: Orchestration Architecture

OmniMind operates on a three-tier routing logic, ensuring that every query is handled by the model—or combination of models—best suited for the task.

### 1. Web Discovery Pool (Tinyfish)
*   **Autonomous Research**: Executes deep-web discovery to ground responses in real-time truth.
*   **Deep Context Extraction**: Utilizes specialist "Fetch" agents to parse raw HTML/Markdown from the digital landscape.
*   **Citation Integrity**: Integrated **Nexus Citation Carousel** ensures every fact is backed by a verifiable source.

### 2. Visual Reasoning Pool (Moondream)
*   **Visual Grounding**: High-fidelity image understanding and spatial reasoning.
*   **Multimodal Synthesis**: Combines visual data with text-based documents for comprehensive analytical outputs.

### 3. Infinite Intelligence Pool (Specialist Ensemble)
*   **Orchestrated Reasoning**: Dynamically dispatches logic to GPT-4o, Gemini 1.5 Pro, or Llama 3 based on domain expertise (Coding, Creative, Mathematical).
*   **SSE Streaming Architecture**: Ultra-low latency response delivery with real-time reasoning chain visibility.

## 📄 Universal Document Ingestion

OmniMind features a robust, multimodal ingestion pipeline capable of processing complex datasets and technical documents without context loss.

*   **Supported Formats**: `.pdf`, `.txt`, `.md`, `.csv`, and high-resolution images.
*   **Premium Presentation**: Utilizing **Hugeicons** and **Nexus UI**, documents are rendered as sleek, identifiable cards within the chat history.
*   **Context Injection**: Decodes and injects file contents directly into the orchestration prompt for deep analytical reasoning.

## 🎨 Design Philosophy: Liquid Glass

We believe that intelligence should be beautiful. Our design system, **Liquid Glass**, is built on three pillars:

### 1. Ethereal Minimalism
*   **Cinematic Backgrounds**: A fixed, high-fidelity landscape GIF creates a stationary visual anchor, allowing content to glide effortlessly over it.
*   **Luminosity Blending**: Backgrounds are blended with luminosity filters to prevent visual fatigue while maintaining an immersive atmosphere.

### 2. Typographic Precision
*   **Outfit (Display)**: A modern, geometric sans-serif used for headers to convey a sense of future-forward innovation.
*   **Geist Mono (Technical)**: A custom mono-spaced font used for technical metadata, reasoning chains, and code blocks.

### 3. Reasoning Transparency
*   **The Reasoning Chain**: A real-time, expandable "Thinking" log that reveals the Brain Router's internal decision-making process, tool selection, and grounding steps.

## 🚀 Deployment & Local Setup

### 🔑 Environment Configuration
The platform requires active API keys for the discovery and vision layers.
```env
TINYFISH_API_KEY=your_key_here
MOONDREAM_API_KEY=your_key_here
```

### 📦 Installation & Bootstrapping
```bash
# Clone the repository
git clone https://github.com/omnimind/orchestrator.git

# Install high-fidelity dependencies
npm install

# Start the discovery server
npm run dev
```
